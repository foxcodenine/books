Using pattern flags

The i modifier is used to perform case-insensitive matching.

The m modifier is used to perform a multiline match.

The m modifier treat beginning (^) and end ($) characters to match the beginning or end of each line of a string (delimited by \n or \r), rather than just the beginning or end of the string.

Note: The m modifier is case-sensitive and will stop the search after the first match. To perform a global, case-insensitive, multiline search, use this modifier together with "g" and "i".

The g modifier is used to perform a global match (find all matches rather than stopping after the first match).

Using the rgx.test method

The test method only returns true or false, based on whether a string matches a pattern or not.

rgx = /hello/;

console.log(rgx.test("hello"));

true

Using the rgx.exec method

The exec function also returns some information about the match.

console.log(rgx.exec("hello"));

["hello", index: 0, input: "hello", groups: undefined]

console.log(rgx.exec("hi"));

null

Using the String.replace method

The replace method will switch the first occurrence, adding the g flag, will replace all the occurrences.

str = 'foo foo';

str.replace(/foo/, 'qux')

qux foo

str.replace(/foo/g, 'qux')

qux qux

Using the String.search method

The sarch method find the (zero-based) index of the first match in a string.

let str3 = "hello world!"

console.log(str3.search(/world/));

6

Using the String.match method

The match method returns the same output as the exec function we saw earlier when there was no g flag (it includes the index and input properties), but returned a regular Array of all the matches when the g flag was set.
