Chapter 2   The Basics

Matching a wild card character

A period in Regex will match any character except a new line, so it can include letters, numbers, symbols, and so on.

let str, rgx;

str = '123 1b3 1 3 133 321'

rgx = /1.3/g;

console.log(str.match(rgx));

>(4) ["123", "1b3", "1 3", "133"]

Matching digits

The backslash d or digit special character will match any character between 0 to 9.

str = '123 1b3 1 3 133 321'

rgx = /1\d3/g;;

console.log(str.match(rgx));

>(2) ["123", "133"]

Matching alphanumeric chars

The backslash w is a word special character. It will match the underscore character, numbers, or any of the 26 letters of the alphabet (in both lowercase as well as uppercase letters).

str = '123 1b3 1 3 133 321'

rgx = /1\w3/g;

console.log(str.match(rgx));

>(3) ["123", "1b3", "133"]

Negating alphanumeric chars and digits

If you want the negated versions of \d and \w, you can just use their uppercase counterparts, \D and \W.

str = '123 1b3 1 3 133 321'

rgx = /1\D3/g;

console.log(str.match(rgx));

>(2) ["1b3", "1 3"]

rgx = /1\W3/g;;

console.log(str.match(rgx));

>["1 3"]
